Ability

data class Ability(val ability: AbilityX, val is_hidden: Boolean, val slot: Int)

Dataklasse die een Pokémon-vaardigheid (ability) representeert, zoals ontvangen van een netwerkrespons.

Deze klasse maakt deel uit van de datastructuur die gebruikt wordt voor het deserializeren van responsgegevens van de Pokémon API. Het bevat details over een specifieke vaardigheid van een Pokémon.
