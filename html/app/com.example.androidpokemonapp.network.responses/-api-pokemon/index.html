ApiPokemon

data class ApiPokemon(val abilities: List<Ability>, val height: Double, val id: Int, val moves: List<Move>, val name: String, val types: List<Type>, val weight: Double)

Dataklasse die een Pokémon representeert zoals die door de externe API wordt geleverd.

Deze klasse bevat gedetailleerde informatie over een specifieke Pokémon, inclusief vaardigheden, ervaringspunten, vormen, spelindices, grootte, en meer.
