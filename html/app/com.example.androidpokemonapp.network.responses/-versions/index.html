Versions

data class Versions(val generationi: GenerationI, val generationii: GenerationIi, val generationiii: GenerationIii, val generationiv: GenerationIv, val generationv: GenerationV, val generationvi: GenerationVi, val generationvii: GenerationVii, val generationviii: GenerationViii)

Data klasse die informatie bevat over de verschillende generaties van Pokémon-games.

Deze klasse bevat subklassen voor elke generatie, zoals GenerationI, GenerationII, enzovoort, die elk op hun beurt informatie bevatten over de visuele representaties van Pokémon in die generatie. Het biedt een georganiseerde manier om toegang te krijgen tot de visuele gegevens van Pokémon in verschillende game-generaties.
